# Sigma 3 : Castelõo Team Project

Our design presents a scalable electronic system designed to enhance fan engagement, accessibility, and safety within football stadiums. The system architecture consists of very low-cost seat nodes connected via a wired CAN bus to local section hubs, which communicate wirelessly with a central control server over existing stadium Wi-Fi infrastructure.
Each base-tier seat node provides occupancy detection, LED bars for stadium-wide visual entertainment, simple user inputs for interactive fan participation, an emergency button for fans in need of medical assistance, and an audio jack delivering real-time match commentary for spectators who prefer an assisted listening experience. Coordinated lighting effects and crowd voting enable various interactions such as goal celebrations, crowd waves, and player-of-the-match polls. A dedicated safety mode allows the system to override entertainment features, providing high-visibility evacuation guidance through lighting cues. 
The platform is designed as a tiered system, allowing premium seats to incorporate additional hardware such as a 7-inch touch display that provides a more elevated and interactive experience, while maintaining the same robust, low-cost base configuration. This design aims to be modular and keep costs down, making the product more competitive.
